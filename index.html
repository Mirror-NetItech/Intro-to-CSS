Introduction To CSS

*Alright, Let's better understand what the CSS really entails.


  • *What CSS* :

    CSS is a style language that defines layout of HTML documents. CSS is an acronym which means Cascading Style Sheets.

    CSS is the recommended way to control the presentation layer in a web document. CSS allows you to create rules that control the way that each individual box (and the contents of that box) is presented. A style sheet is a grouping of formatting instructions that can control the appearance of many HTML pages at once.
    The technology behind style sheets is called CSS which defines style constructs such as fonts, colors and positions which are used to describe how information on a web page is formatted and displayed.

    CSS Solved a Big Problem
    HTML was never intended to contain tags for formatting a web page!
    HTMLwas created to describe the content of a web page, like:
    <h1>This is a heading </h1>
    <p>This is a paragraph. </p>
    When tags like <font>...</font>, and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.
    To solve this problem, the World Wide Web Consortium W3C created CSS. CSS removed the style formatting from the HTML page!

    • Benefits Of Using CSS
  • CSS saves time: Writing CSS code is much faster than formatting each HTML element individually. With CSS, you can define a style once and apply it to multiple elements, saving you time and effort.
  • It Helps pages to load faster: CSS files are typically smaller than HTML files, which means they load faster. This is because CSS only contains styling information, whereas HTML contains both content and structure. Faster loading times improve user experience and search engine optimization (SEO).
  • Superior styles to HTML: CSS offers more advanced and flexible styling options compared to HTML. With CSS, you can create complex layouts, animations, and effects that would be difficult or impossible to achieve with HTML alone.
  • Multiple Device Compatibility: CSS allows you to write media queries that adapt your styling to different devices and screen sizes. This ensures your website looks great on desktops, laptops, tablets, and mobile phones.
  • Global web standards: CSS follows web standards set by the World Wide Web Consortium W3C, ensuring consistency across different browsers and devices. This means your website will look and function as intended, regardless of where it's accessed.
  • The separation of content from presentation: enables you to separate your website's content (HTML) from its presentation CSS. This makes it easier to maintain and update your website, as changes to the content don't affect the presentation and vice versa.
  • Separate styling rules for different output media CSS allows you to define different styling rules for various output media, such as print, screen, or speech. This means you can optimize your website's appearance for different contexts, like printing or screen reading.

    • Types Of Style Sheets.
    • External Style Sheet: An external style sheet is ideal or use when you want to apply style to many pages.
      These are separate files Eg: ( style.css ) that contain all the styling rules and are linked to multiple HTML documents. This allows for easy maintenance and consistency across a website.
      And are used to allow people to format and recreate their webpages on an entirely different document, that means you will have your HTML document on a different file while the CSS document too will be on a different file, thereby providing you a much cleaner workspace.
      Note: Each page or HTML document must link to the external style sheet using the <link> tag. This <link> tag goes inside the <head>...</head> section:
      Eg: 
        <!DOCTYPE html>
  <html lang="en">
  <head>
    <title>External Style Sheet</title>
    <!-- The below is the link tag that link my external 
     style sheet named " mystyle.css "-->
    <link rel="stylesheet" type="text/css" href="mystyle.css">
    Or
    You can also use the @import method as shown below:
    <style type="text/css">@import url(mystyle.css)</style>
  </head>
  <body>
  
  
  </body>
  </html>
      Note:
      Type- The type="" attribute specifies the style sheet language.
      Href- The href="" attribute specifies the path to the CSS file.
      rel- The rel="" attribute specifies relationship between the Html page and the css file it is linked to.
      Check out the steps to create
      and link your external style sheet

      Open your text editor and create two new file and name them using myhtmlfile.html and also myexternalfile.css and save them, so our file extensions here are .html and .css

      Open the HTML file and make sure you have the basic structure of HTML or HTML5 .

      Now let's link our myhtmlfile.html document with the external style sheet named myexternalfile.css:
      Inside the <head>...</head> section under the title tags type the following codes there:
      <link rel="stylesheet" type="text/css" href="myexternalfile.css">
      Alright, now we have linked our Html file with our external CSS file.

    • Internal Style Sheet: Internal style sheet is used used when you wanted to add a small change in the single tag and also is applied to only one HTML document.
      These are embedded within an HTML document using the <style>...</style> tag, inside the <head>...</head> section under the title tag.
      Eg: 
        <!DOCTYPE html>
  <html lang="en">
  <head>
    <title>Internal Style Sheet</title>
    <!-- The below is the link tag 
    that link my Internal style sheet-->
     <style type="text/css" media="all">
      body{
        background-color:yellow;
        }
        p{
        color:blue;
        }
     </style>
  </head>
  <body>
   <p>This is an Internal Style Sheet.</p>
  
  </body>
  </html>
      Note: The media="" attribute specifies the device the document will be displayed on with values such as ( screen, ttv, tv, projection, handheld, print, braille, aural and all ).
    • Inline Style Sheet: An inline style sheet is used if a are applied directly to an HTML element using the style="" attribute and it only affect that individual element. . The style"" attribute can contain any CSS property. Eg: 
        <!DOCTYPE html>
  <html lang="en">
  <head>
    <title> Style Sheet</title>
  </head>
  <body>
    <!-- The below is the inline style 
  sheet affecting the color, font size 
  and the left margin of the paragraph-->
    <p style="font-size:110%; color:blue; margin-left: 20px;">
      This is a paragraph
    </p>
  </body>
  </html>
